## Полный перечень функциональности (охраняемый состав продукта)

### Команды агентов ИИ (мультимодельная оркестрация)

- **10 стратегий оркестровки:** последовательная, круговая, параллельная, иерархическая, голосование, дебаты, цепочка мыслей, умный маршрутизатор, MapReduce, экспертная панель.
- **12 ролей агентов:** лидер, работник, исследователь, аналитик, программист, рецензент, составитель резюме, переводчик, творческий, проверяющий факты, маршрутизатор, пользовательский менеджер.
- **14 инструментов агента:** веб-поиск, чтение/запись файлов, выполнение кода, суммирование, перевод, анализ изображений, расшифровка аудио, запрос/сохранение памяти, делегирование, калькулятор, разбор JSON, регулярные выражения.
- **Права доступа агента:** интернет, камера, микрофон, экран, файлы, инструменты, делегирование, вето, STM, LTM, общая память.
- **Ресурсы агента:** лимит RAM, ядра CPU, токены, температура, порядок выполнения.
- **Настройки группы:** макс. раунды, тайм-аут, 6 режимов разрешения конфликтов, порог консенсуса, декомпозиция задач, петли обратной связи.
- **Условия срабатывания:** всегда, ключевое слово, делегирование, раунд, сбой, пользовательский.
- **Монитор оркестровки** в реальном времени (Agent Studio).

### Система памяти ИИ

- **STM (краткосрочная):** 5–200 сообщений на чат, TTL, авто-суммирование.
- **LTM (долгосрочная):** сохранение фактов между сессиями с оценкой важности.
- **3 уровня доступа:** `CHAT_ONLY`, `MODEL_SHARED`, `GLOBAL`.
- Перенос памяти между чатами и моделями, консолидация STM→LTM, семантический поиск.

### Системные инструкции и вводные

- Системная подсказка для каждого сообщения в чате.
- Глобальные настройки по умолчанию (префикс, суффикс, скрытый контекст, время, локаль).
- Переопределения для каждого агента и чата.

### Сеть и безопасность

- Три режима изоляции: полная, только API (белый список), открытый.
- Поиск DuckDuckGo с панелью активности в реальном времени.
- Монитор всех HTTP-запросов, контроль прав на чат и модель.
- Защита от prompt injection, шифрование настроек/памяти (опции).

### Настройки (79+ параметров в 9 категориях)

1. Память и ОЗУ  
2. ЦП и многопоточность  
3. Настройки вывода по умолчанию  
4. Системные инструкции  
5. Система памяти ИИ  
6. Интернет и веб  
7. Разрешения по умолчанию  
8. Внешний вид (Material 3, темы)  
9. Расширенные + агенты  

Дополнительно: инновационные технологии (нейропластичная память, когнитивный балансировщик, квантовые слои контекста и др.), производительность (турбо, KV-cache, pipeline), устройства (камера, микрофон, экран).

### Прочее

- Несколько чатов с независимыми правами.
- 9-шаговый онбординг, всплывающие подсказки.
- Браузер Hugging Face + загрузка моделей (GGUF, ONNX, Safetensors, PT, BIN, GGML).
- Многорежимный ввод (камера, микрофон, файлы).
- Локализация RU/EN.
- Кроссплатформенность: Windows и Linux.
