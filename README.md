# NeuroForge

**NeuroForge** — кроссплатформенное приложение (Windows / Linux) для локального запуска нейросетей с Hugging Face и пользовательских моделей.

## Возможности

- **Локальный инференс** — GGUF, ONNX, Safetensors, PT, BIN, GGML
- **Изоляция сети** — полная, только API (белый список), или открытый доступ
- **Память** — краткосрочная (STM) и долгосрочная (LTM) с переносом между чатами и моделями
- **Глобальные вводные** — инновационная настройка: текст, добавляемый к каждому сообщению
- **Команды агентов** — группы нейросетей с ролями и разными правами
- **Устройства** — камера, микрофон, захват и расширение экрана
- **Сетевой монитор** — видимость всех запросов нейросети в интернет
- **Настройки** — RAM, CPU-ядра, GPU, потоки, батч, и десятки других параметров
- **Сброс по умолчанию** — восстановление всех настроек одной кнопкой
- **Локализация** — русский и английский, выбор языка при первом запуске
- **Справка и обучение** — встроенный onboarding и документация по всем настройкам

## Сборка

### Требования

- Node.js 22+
- Rust stable
- Linux: `libwebkit2gtk-4.1-dev`, `libayatana-appindicator3-dev`, `librsvg2-dev`, `patchelf`

### Команды

```bash
npm install
npm run lint
npm run tauri:dev      # режим разработки
npm run tauri:build    # production-сборка
```

Артефакты: `src-tauri/target/release/bundle/`

## GitHub Actions

Workflow `.github/workflows/build.yml` собирает артефакты для Linux и Windows при каждом push/PR. Релизы создаются при push тега `v*`.
